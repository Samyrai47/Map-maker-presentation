# API

Авторизация и аккаунты

POST /auth/register - регистрация пользователя. Тело: {email, password, username}

POST /auth/login - вход в аккаунт (аутентификация). Тело: {email, password}

Работа с картами

GET /maps - получение списка карт пользователя.

GET /maps/{id} - получение данных конкретной карты.

PUT /maps/{id} - обновить карту.

DELETE /maps/{id} - удалить карту.

Экспорт карт

POST /maps/{id}/export/png - сгенерировать PNG.

POST /maps/{id}/export/pdf - сгенерировать PDF.

Музыкальный плеер

GET /music - получить список треков.

GET /music/{id}/stream - передача трека.

Библиотека пресетов

GET /assets/objects - получить список доступных предметов.
